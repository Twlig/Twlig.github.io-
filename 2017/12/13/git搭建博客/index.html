git搭建博客

第一次搭建自己的博客超激动呢。之前一直想搭建自己的博客来记录一下自己的学习成果，但是看到那些用worldpress搭建，总觉得很难，直到现在在简书上看到了别的大佬写的文章，终于用git搭建了自己的博客。那接下来我就把我的搭建过程详细的写下来了。

搭建这个博客需要用到的内容：

1、git和github：

下面我来安利一波git教程:git教程
还有github教程
这两个教程都是比较好的教程，简洁详细。

2、node.js

node.js教程
node.js的其实在搭建博客里只是主要用到了里面的npm这个东西，所以我在上面引用的教程链接只需要看node.js的安装配置。
安装好了node之后。打开运行窗口。如果是windows系统可以用windows键加R打开运行窗口

3、安装hexo

安装好了node.js之后，安装hexo就很简单啦，打开windows运行窗口一行命令就可以安装hexo。

4、markdown

咱们使用markdown来编写博客所以，我们需要安装markdown编译器。markdown的编译器很多可以自己下载喜欢的。我用的是markPad。除了下载编译器之外，还需要了解一定的markdown的语法，它和word的写文档的用法不同，有一定的格式要求。这篇是认为比较好的文章：markdown语法

正式搭建博客

1、首先创建git账号

根据上面的教程注册git账号，下面这个地址是github的注册页面：git注册

2、创建git仓库

我们需要两个仓库，一个git仓库用来存放自己写的博客，一个git仓库用来存放博客里面的图片。用来存放自己写的博客的仓库，命名使用你的git的账号名username，具体格式是username.git.io。咱们为啥要用git来存放图片呢？其实主要是因为我们的博客会用到markdown来写，而在markdown里面放图片巨麻烦，所以用git仓库的形式存放图片然后直接引用就可以了。

下面这个命令也是在运行窗口的cmd，打开控制台里面敲。

4、更改配置

1、安装主题

这个更改配置其实是为了改变页面的布局，在之前hexo init的时候其实已经下载了默认的主体，但是next，样式更为美观。

2、更改配置

打开文件位置username.github.io/_config.yml修改几个键值对，下面把几个必须设置的列出来按需求修改， 还有注意配置的键值之间一定要有空格。.ym;后缀的文件可以markdown，还有代码编译器也行，甚至word都可以。

title: //你博客的名字
author: //你的名字
language: zh-Hans //语言 中文
theme: next //刚刚安装的主题名称
deploy:
type: git //使用Git 发布
repo: https://github.com/username/username.github.io.git // 刚创建的Github仓库

5、写文章

所有基础框架都已经创建完成，接下来可以开始写博客。在filename.github.io/source/_posts下创建你的第一个博客，例如，创建一个名为firstTry.md的文件，用Markdown编译器编写。语法什么的我在前面的内容有讲到。还有在Markdown里插入图片很麻烦，我在这里提供了一种方法就是上传到git仓库。然后右键保存图片地址在用就可以了。
